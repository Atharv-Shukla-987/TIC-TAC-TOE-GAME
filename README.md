# Tic-Tac-Toe Game

A classic and  interactive  game built with simple user experience in mind. This project demonstrates basic turn-based mechanics and win-condition algorithm.
🚀 Features

Two-Player Mode: Play against a friend (locally)

clean UI

 Win Detection: Automatically detects Horizontal, Vertical, and Diagonal wins.
 trun detection is present there  and  Prevents players from choosing occupied squares or invalid positions.

## 🛠️ Built With

    HTML
    CSS
    Javascript

## 🎮 How to Play

    Go to https://tic-tac-toe-game-murex-seven.vercel.app/
    and u can start playing ith ur friend
    The first player to make three x or O in a line wins
    if all squares are full  You can reset the game using reset buttton
